GROUP 39

1. Open the FSD-Project folder in your desired editor(most preferably VS Code).
2. Go to FSD-Project folder and run npm install which will install all the neccesary packages
required. Now npm run server which actually runs nodemon as backend server or you can run node server.js command also. Now it will open loclahost:5000 in your local browser.
3. Now go to ecommerce folder and run npm install which will install all the required packages. Now run npm start which will run as localhost:3000 in your browser and it will land on the starting page of our page.
4. Done !
